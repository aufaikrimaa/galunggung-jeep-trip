## Project Summary
A minimal, stylish company profile website for Galunggung Jeep Trip, an offroad tour provider at Gunung Galunggung.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Google Fonts (Syne, Outfit)
- Framer Motion (Implicitly via Tailwind animations)

## Architecture
- `src/app/page.tsx`: Single-page layout with all sections.
- `src/app/layout.tsx`: Font configuration and root layout.
- `src/app/globals.css`: Custom animations and Tailwind configuration.

## User Preferences
- Minimal content, avoid clutter.
- Stylish and "cool" aesthetic.
- Modern typography (Syne for headings, Outfit for body).
- Dark theme with emerald/safety-orange accents.

## Project Guidelines
- Keep typography bold and visually appealing.
- Use high-quality imagery (placeholders/Unsplash for now).
- Maintain a high-contrast, moody color palette.
- No comments in code.
- Functional components only.

## Common Patterns
- Horizontal scrolling galleries.
- Fixed glassmorphism navigation.
- Large, tracking-tight headings.
